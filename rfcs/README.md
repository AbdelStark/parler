## RFC process

1. Copy an existing RFC as a template
2. Fill in Abstract, Motivation, Design, Alternatives, Open Questions
3. Open a PR for discussion
4. Status moves: `Draft` → `Accepted` → `Implemented` | `Withdrawn`
